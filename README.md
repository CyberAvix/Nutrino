<h1>Nutrino</h1>

Predložena veb aplikacija omogućava korisnicima da jednostavno analiziraju nutritivne vrednosti namirnica, uključujući količine proteina, vitamina, kalorija i drugih relevantnih parametara. Sistem koristi pregledne vizualizacije putem D3.js biblioteke kako bi podaci bili lako razumljivi i intuitivni. Aplikacija je namenjena studentima, sportistima, nutricionistima i svim korisnicima koji žele brzu i pouzdanu procenu sastava hrane. Fokus je na jednostavnom i modernom interfejsu koji korisniku daje precizne informacije bez komplikacija.
